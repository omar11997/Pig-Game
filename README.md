# Pig-Game

he Pig Game project! This is a simple and fun web-based game where players take turns rolling a dice to accumulate points. The goal of the game is to be the first player to reach a certain score or a predefined winning score.

Features

Two Players: The game supports two players who take turns rolling the dice.
Rolling Dice: Players can roll a dice, and the rolled number gets added to their current round score.
Hold: Players can choose to "hold," which means their current round score is added to their total score, and the turn switches to the other player.
Switching Players: The active player changes after each turn, and the round score resets to zero.
Winning: The first player to reach the predefined winning score wins the game.
New Game: Players can start a new game at any time, resetting the scores and player statuses.


Technologies Used
HTML
CSS
JavaScript (ES6+)


How to Play

Open the project in a web browser.
Enter the predefined winning score (e.g., 100 points) and start the game.
Players take turns by rolling the dice.
Players can decide to hold their current score or keep rolling.
The first player to reach the predefined winning score wins!
